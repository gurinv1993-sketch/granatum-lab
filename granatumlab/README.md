### Текущий этап проекта (MVP в разработке):
- **Маршрутизация и архитектура**: Реализована система URL-паттернов и иерархия представлений (Views).
- **Frontend-шаблонизация**: Создана модульная система шаблонов с использованием наследования (base.html). Настроена навигация по разделам сервиса.
- **Проектирование БД (ORM)**: Разработана схема базы данных (SQLite). Реализованы модели для продуктов (с детальным химическим составом), категорий и рецептов.

### Roadmap (в процессе реализации):
- Наполнение базы данных нутрициологическими данными.
- Разработка алгоритма сопоставления ингредиентов в наличии с базой рецептов.
